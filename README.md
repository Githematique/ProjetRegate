# ProjetRegate

# Installation

1. Clone repo

2. Change to directory
```sh
$ cd ProjetRegate
```
3. Install dependencies
```sh
composer install
```
4. Create .env file
```sh
create .env file in the directory
```
```sh
copy content from .env.exemple to .env
```
5. Generate the key
```sh
php artisan key:generate
```
6. Install Node modules
```sh
$ npm install
```
7. Start the application
```sh
$ php artisan serve
```
